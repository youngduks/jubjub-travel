# 줍줍여행사 (jubjub-travel)

형님 본인 도구 — 평소 자주 가는 8개 도시 항공권 검색 3 click.

## 도시 (dogfooding default)

- 🇰🇷 제주
- 🇯🇵 도쿄·오사카·후쿠오카·오키나와
- 🇻🇳 다낭·푸꾸옥
- 🇹🇭 방콕

## Local dev

```bash
npm install
npm run dev
```

→ http://localhost:3000

## Deploy

Vercel CLI 또는 GitHub push → Vercel auto-deploy. 도메인 X → `*.vercel.app` subdomain 사용.
